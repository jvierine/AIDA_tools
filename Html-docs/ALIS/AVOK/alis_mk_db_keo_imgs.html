PURPOSE

ALIS_MK_DB_KEOS - Update ALIS Keogram-database

SYNOPSIS

function ok = alis_mk_db_keo_imgs()

DESCRIPTION 

 ALIS_MK_DB_KEOS - Update ALIS Keogram-database
   
 Calling:
 ok = alis_mk_db_keos;
 Currently working under the assumption that the images are under
 /alis/stdnames and in a directory tree with the following
 structure:
 /alis/stdnames/YEAR/MONTH/DAY/
 From the DAY directory files for all images taken that day is
 searched for with unix FIND.

CROSS-REFERENCE INFORMATION

This function calls:
  • alis_imgs2keos ALIS_IMGS2KEOS - make overview-keograms
  • inimg INIMG - reads and pre-process an image.
  • typical_pre_proc_ops TYPICAL_PRE_PROC_OPS - Typical ALIS-fits preprocessing options
