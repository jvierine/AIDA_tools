DESCRIPTION 

 STARINT evaluation of a 2D gaussian.
 
 STARINT calculates a 2D gaussian on a matrix located
 between (xmin,xmax) and (ymin,ymax) the maxintensity
 and the spread of the gaussian is given in the array
 FV.
