mean2

PURPOSE

MEAN2 Compute mean of matrix elements.

SYNOPSIS

function y = mean2(x)

DESCRIPTION 

MEAN2 Compute mean of matrix elements.
   B = MEAN2(A) computes the mean of the values in A.

   Class Support
   -------------
   A is an array of class uint8 or double. B is a scalar of
   class double.
