ALIS_pre_pos_vv_plot

PURPOSE

ALIS_pre_pos_vv_plot - plot the visible volume of ALIS

SYNOPSIS

function hndl = ALIS_pre_pos_vv_plot(configNR)

DESCRIPTION 

 ALIS_pre_pos_vv_plot - plot the visible volume of ALIS
  pre-defined camera rotations. 

 Calling:
   hndl = ALIS_pre_pos_vv_plot(configNR)
 Input:
   configNR - ALIS configuration number for pre-defined:
               1 - Surveilance
               2 - Magnetic zenith
               3 - South
               4 - Core
               5 - EAST-West
               6 - North
               7 - EISCAT
               8 - Heating
               9 - EISCAT-2
 Output:
   hndl - handle to the lines of the plotted visible-volume
   borders 
 
 Example:  
   H = ALIS_pre_pos_vv_plot(8);
 
 See also: ALISstdpos_visvol

CROSS-REFERENCE INFORMATION

This function calls:
  • ALISstdpos_visvol ALISSTDPOS_VISVOL - Display ALIS f-o-v ontop of topographic map
